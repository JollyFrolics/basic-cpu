# A2SM

A basic circuit design based on the Basic Computer by Morris Mano. This is a very simple CPU design.
I have made some adjustments to it. There is a stack based function call system. So, multiple function calls can be made by a program.
The previous system tracked the return address in the beginning of a function which means that recursion is not possible.
